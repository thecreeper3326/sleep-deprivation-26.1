# Sleep Deprivation 26.1.x
Sleep Deprivation Mod by TheCreeper3326

As players go without sleep, they receive negative effects and their items in their inventory will start to switch places.

This fork is for all 26.1.X minecraft versions

For more mod info go look at the Modrinth page:
https://modrinth.com/mod/johnseagull-sleep-deprivation/
